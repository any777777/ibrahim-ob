# Ibrahim Obaidat — Portfolio

A bilingual Arabic-first portfolio for Ibrahim Obaidat’s website design, hosting, and management service in Jordan.

## What is included

- Layered interactive showcase for seven selected Vercel projects
- Arabic and English interface with correct RTL/LTR direction
- Mobile-first responsive layouts
- Interactive project price estimator
- Pre-filled WhatsApp enquiries sent to `+962 79 861 3275`
- Search metadata, sitemap, robots rules, and reduced-motion support

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
npm audit --audit-level=moderate
```

## Project notes

- `PRODUCT.md` records product positioning and audience decisions.
- `DESIGN.md` records the visual direction and interaction principles.
- `PROJECTS.md` records the initial portfolio curation assumptions.

The public production deployment is hosted on Vercel.
